# Get some z's

{% tabs %}
{% tab title="Prompt" %}
{% hint style="info" %}
Can you instruct your computer to print the letter `z` over 100 times?

How about 1000?
{% endhint %}
{% endtab %}

{% tab title="Solution: 100" %}
```python
for i in range(100):
    print('z')
```
{% endtab %}

{% tab title="1000" %}
```python
for i in range(1000):
    print('z')
```
