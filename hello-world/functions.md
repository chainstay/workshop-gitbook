# Functions

```python
print()
```

## Anatomy of a function

`name` `(` `arguments` `)`

### Some examples

```python
print('Hiiiii')  # name: print, arguments: 'Hiiiii'

type(1)  # name: type, arguments: 1

print(1, 2, 3) # name: print, arguments: 1, 2, and 3
```
